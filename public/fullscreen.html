Is your game stuck in a small window, low resolution, or showing black bars on the sides?

Here’s a simple way to fix it using the unnamedsoldier.cfg config file.

This works perfectly on MOHAA Revival Edition (which you can download on our website) and is tested on Windows 10 and 11.

Step 1: Find Your Screen Resolution

  • Right-click your desktop → choose Display settings
  • Look for Display resolution (example: 1920 x 1080)
  • Write down or remember these two numbers: width and height

Step 2: Edit unnamedsoldier.cfg

For Spearhead go to this folder:

C:\Games\MOHAA\mainta\configs\

For Breakthrough the correct folder is:

C:\Games\MOHAA\maintt\configs\

  1. Find the file unnamedsoldier.cfg
    If it doesn't exist, just run the game once — it will be created automatically.
  2. Right-click the file → Open with Notepad
  3. Scroll to the bottom and add or replace the following lines:
seta r_mode "-1"              // Enables custom resolution mode
seta r_customwidth "1920"     // Set this to your screen width
seta r_customheight "1080"    // Set this to your screen height
seta r_fullscreen "1"         // Enables full screen

Example: For a 1920×1080 monitor, copy the settings exactly.
If your screen is for example 1360×768, change those numbers accordingly.

Step 3: Save and Launch the Game

  • Click File > Save in Notepad
  • Launch Medal of Honor: Spearhead
  • It should now run in true fullscreen using your correct resolution

That’s It!

This fix solves:

  • Fullscreen not working
  • Wrong or low resolution
  • Widescreen issues on Windows 10 / 11

Tested and working on MOHAA Revival Edition.
Feel free to share this guide with other players!

Ваша игра запускается в маленьком окне, с низким разрешением или с чёрными полосами по бокам?

Вот простой способ исправить это с помощью файла unnamedsoldier.cfg.

Этот способ отлично работает в MOHAA Revival Edition (который вы можете скачать на нашем сайте) и протестирован на Windows 10 и 11.

Шаг 1: Узнайте разрешение вашего экрана

  • Кликните правой кнопкой мыши по рабочему столу → выберите Параметры экрана
  • Найдите Разрешение экрана (пример: 1920 x 1080)
  • Запомните/запишите ширину и высоту экрана

Шаг 2: Отредактируйте unnamedsoldier.cfg

Для Spearhead перейдите в папку:

C:\Games\MOHAA\mainta\configs\

Для Breakthrough правильная папка:

C:\Games\MOHAA\maintt\configs\

  1. Найдите файл unnamedsoldier.cfg
    Если файла нет — просто запустите игру один раз, он создастся автоматически.
  2. Кликните правой кнопкой → Открыть в Notepad
  3. Прокрутите вниз и добавьте/замените строки:
seta r_mode "-1"              // Enables custom resolution mode
seta r_customwidth "1920"     // Set this to your screen width
seta r_customheight "1080"    // Set this to your screen height
seta r_fullscreen "1"         // Enables full screen

Пример: для монитора 1920×1080 скопируйте настройки как есть.
Если у вас, например, 1360×768 — измените цифры соответственно.

Шаг 3: Сохраните и запустите игру

  • В Notepad нажмите File > Save
  • Запустите Medal of Honor: Spearhead
  • Игра должна работать в настоящем полноэкранном режиме с правильным разрешением

Готово!

Этот способ решает:

  • Не работает полноэкранный режим
  • Неверное или низкое разрешение
  • Проблемы широкоформатного режима на Windows 10 / 11

Проверено и работает на MOHAA Revival Edition.

Zit je spel vast in een klein venster, met een lage resolutie of zwarte balken aan de zijkanten?

Hier is een eenvoudige manier om dit op te lossen met behulp van het configuratiebestand unnamedsoldier.cfg.

Dit werkt perfect op MOHAA Revival Edition (die je kunt downloaden op onze website) en is getest op Windows 10 en 11.

Stap 1: Zoek je schermresolutie

  • Klik met de rechtermuisknop op je bureaublad → kies Beeldscherminstellingen
  • Zoek naar Beeldschermresolutie (bijvoorbeeld: 1920 x 1080)
  • Noteer of onthoud deze twee getallen: breedte en hoogte

Stap 2: Bewerk unnamedsoldier.cfg

Voor Spearhead ga je naar deze map:

C:\Games\MOHAA\mainta\configs\

Voor Breakthrough is de juiste map:

C:\Games\MOHAA\maintt\configs\

  1. Zoek het bestand unnamedsoldier.cfg
    Als het niet bestaat, start het spel dan eenmaal op — het wordt automatisch aangemaakt.
  2. Klik met de rechtermuisknop op het bestand → Openen met Kladblok
  3. Scroll naar beneden en voeg de volgende regels toe of vervang ze:
seta r_mode "-1"              // Enables custom resolution mode
seta r_customwidth "1920"     // Set this to your screen width
seta r_customheight "1080"    // Set this to your screen height
seta r_fullscreen "1"         // Enables full screen

Voorbeeld: kopieer de instellingen exact voor een monitor met een resolutie van 1920×1080.
Als je scherm bijvoorbeeld 1360×768 is, pas deze getallen dan dienovereenkomstig aan.

Stap 3: Opslaan en het spel starten

  • Klik op Bestand > Opslaan in Kladblok
  • Start Medal of Honor: Spearhead

Deze oplossing verhelpt:
