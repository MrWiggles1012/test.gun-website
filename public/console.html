Console

Enable the in-game console, open it quickly, and use the mini console.

Translations Available:
English | Arabic | German | Russian | Armenian | Spanish | Dutch



Besides unnamedsoldier.cfg, the in-game console is one of the most important tools for changing settings.

Enable the Console

You can enable it in two ways:

  • Menu → Options → Advanced → enable/tick Console
  • Or via unnamedsoldier.cfg by adding/changing: seta ui_console "1"

After enabling, you can open the console with the key below Escape (usually ` or ~).

Mini Console

If you want a small mini console in the top-right corner that shows messages without opening the full console, type: 

ui_minicon 1

To disable the console/mini-console, set the values back from 1 to 0.

Annoying Console Errors

The console can show many harmless error messages that don’t affect gameplay but can be annoying. (Fixes are usually listed in the guide’s appendix/extra section.)

Naast unnamedsoldier.cfg is de in-game console één van de belangrijkste tools om instellingen aan te passen.

Console inschakelen

Je kunt de console op twee manieren aanzetten:

  • Menu → Options → Advanced → vink Console aan
  • Of via unnamedsoldier.cfg door toe te voegen/aan te passen: seta ui_console "1"

Daarna kun je de console openen met de toets onder Escape (meestal ` of ~).

Mini console

Wil je een kleine mini console rechtsboven die berichten toont zonder de volledige console te openen? Typ: 

ui_minicon 1

Zet de waarden terug van 1 naar 0 om het weer uit te zetten.

Irritante console errors

De console kan veel onschuldige foutmeldingen tonen die de game niet beïnvloeden maar wel irritant zijn. (Fixes staan meestal in een appendix/extra sectie van de guide.)
